Membuka file CSV di Microsoft Office Excel

  1. Buka data yang telah diunduh menggunakan Microsoft Office Excel
  2. Jika pengaturan komputer anda sudah diset dengan separator koma (,), data akan otomatis terpisah per kolom
  3. Jika tidak, blok kolom pertama kemudian klik menu tab "data" dan pilih fungsi "text to columns"

    4. Gambar 1 Blok Kolom Pertama

    Gambar 2 Klik Menu Tab Data -> Text To Column

  4. Pilih option "Delimited", kemudian klik "next"
  5. Pilih "delimiters" yang tersedia (karena pada file contoh menggunakan tanda "," maka ceklis optiion "comma" kemudian klik "next"

    6. Gambar 3 Menu Delimiters

  6. Klik "finish"
